# Idade no Ano

O script principal de vocês deve estar no arquivo `main.py`.

## 📝 Instruções 📝

No `main.py`, escreva instruções para receber os valores abaixo na seguinte ordem:

1. receber do usuário um número que é o ano atual.
2. receber do usuário sua idade atual.
3. receber do usuário outro número representando outro ano.
4. receber do usuário o seu nome.

A ordem de recebimento dos dados é importante para o funcionamento correto dos testes automáticos.

O programa deve fazer os cálculos para imprimir na tela a frase `Fulano, no ano de XXXX você terá YYYY anos`, com os valores corretos.

## 🧑‍💻 Exemplo de Execução 🧑‍💻

Usuário digita:

```
2010
15
2023
Fulano
```

O programa deve imprimir na tela:

```
Fulano, no ano de 2023 você terá 28 anos
```
